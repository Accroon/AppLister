IF USING LOWER VERSION THAN 3.0 THEN DO THE FOLLOWING:

Run the app and go to task schedule >> Create a task >> {name the task} >> go to Triggers >> Click 'New'  >> at the top select 'at log in' or 'at startup' >> go to Actions >> Click 'New'  >> Browse the AppLister.exe >> Apply 


for testing to see if the app runs, click on 'run task' if you see AppLister there in task manager; then its running.


Press 'Ctrl + T" to terminate the app


Edit the text that gets created when you first run it, to set more exe locations to display in the lister.
